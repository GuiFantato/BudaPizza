Projeto feito para o autoatendimento de um cliente com o escopo de uma pizzaria.

Feito seguindo o fluxo principal onde o cliente faz login/cadastro, lê o QRCode de uma mesa,
escolhe e personaliza seu pedido, podendo adicionar ou remover produtos e se dirige ao
pagamento, onde pode pagar com crédito, débito, PIX ou dinheiro e
atribuir um CPF para sua compra.

Utilizamos tecnologias como Prisma ORM e Supabase para o banco de dados e sua organização,
enquanto o frontend foi realizado com React Native e Expo GO.
Mais tecnologias foram utilizadas para complementar o projeto, como o insomnia para
validação da API.
